# 🚗 Where I Parked My Car
## 📱 Projeto Final – Desenvolvimento de Aplicativos Móveis

Este aplicativo foi desenvolvido como Projeto Final da disciplina de Desenvolvimento de Aplicativos Móveis, do Centro Universitário Mater Dei – UNIMATER, utilizando o framework Flutter.

O objetivo do projeto é criar um aplicativo simples, funcional e intuitivo que ajude o usuário a salvar o local onde estacionou o carro e encontrá-lo facilmente depois.

## ⚙️ Funcionalidades

O aplicativo possui as seguintes funcionalidades:
- 📍 Salvar a localização atual do carro
- 🕒 Registrar a data e a hora em que o carro foi estacionado
- 🗺️ Visualizar a localização salva
- 🔙 Navegação entre telas com botão de retorno
